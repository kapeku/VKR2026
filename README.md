# Помощник эксперта: КТЭ ПК под управлением ОС Linux

Веб-приложение для выпускной квалификационной работы. Реализует **гайд-лист** по методике из **главы 3** диплома: комплексное исследование образа жёсткого диска и дампа ОЗУ Linux-системы с Open-Source инструментарием.

## Возможности

- 8 этапов методики (табл. 3.1) с целями, гайдом и контрольными точками
- **Условная логика**: без образа диска отключаются верификация и дисковый анализ; без дампа ОЗУ — этап RAM
- Пошаговый ввод данных эксперта по каждому этапу
- **Отчёт** с ответами на 5 вопросов постановления и сводкой по этапам
- Печать / сохранение в PDF через браузер

## Запуск

```bash
npm install
npm start
```

Откройте http://localhost:3000

## Стек

- Node.js, Express
- EJS (шаблонизатор)
- express-session (данные исследования в сессии)

## Структура

- `data/methodology.json` — этапы, поля, условия показа/пропуска
- `lib/methodology.js` — логика активных этапов
- `lib/report.js` — формирование отчёта
- `views/` — шаблоны EJS

## Этапы методики (глава 3)

1. Подготовительный этап
2. Процессуальная и техническая верификация
3. Комплексный анализ образа диска
4. Исследование дампа ОЗУ
5. Анализ сетевых соединений
6. Механизмы закрепления
7. Корреляция артефактов
8. Интерпретация и документирование
